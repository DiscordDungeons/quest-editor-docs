# Find Index Block

![Find Index Block](../../images/list/find_index.jpg)

Returns the index of the first/last occurrence of the item in the list. Returns 0 if item is not found.

### Generated Code

**first**

```js
var list;

list.indexOf('') + 1;
```

**last**

```js
var list;

list.lastIndexOf('') + 1;
```
